# ART Frontend

## Node Setup
Download and Install Node from the following Website
https://nodejs.org/en/download/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
or
```
npm start
```
### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
